This web application allows users to effortlessly search and analyze historical stock data using real-time APIs. Built with HTML, CSS, and JavaScript for the frontend and Node.js for the backend, the project utilizes the AlphaVantage API to fetch stock data. It is currently hosted on Vercel and version-controlled using GitHub. Please note that the application is still under development, and due to the limitations of the free-tier AlphaVantage API, request frequency is restricted.
